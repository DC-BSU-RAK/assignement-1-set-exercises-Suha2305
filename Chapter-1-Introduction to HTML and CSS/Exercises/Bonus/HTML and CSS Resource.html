HTML & CSS Quick Reference Guide

An essential collection of tags and properties for web development.

HTML Tags

  • !--...-- - Comment

    Adds notes that are not displayed in the browser. Example:

    !-- This is a comment --

  • a - Hyperlink

    Creates clickable links. Example:

    a href="https://example.com">Visit

  • article - Article

    Defines self-contained content. Example:

    
    
    
        
    Article content here.

  • audio - Embedded Sound

    Includes audio playback. Example:

  • b - Bold Text

    Emphasizes text by making it bold. Example:

    b>This is bold.

  • br - Line Break

    Creates a single line break. Example:

    Line 1
    Line 2

  • div - Section

    Defines a container for content. Example:

    Content area

  • h1 to h6 - Headings

    Defines headings of various levels. Example:

    h1>Main Heading

  • img - Image

    Displays images. Example:

    img src="image.jpg" alt="Description"

  • input - Input Control

    Defines an input field. Example:

    input type="text" placeholder="Type here"

  • li - List Item

    Defines items within a list. Example:

    li>Item

  • link - Stylesheet

    Links external CSS files. Example:

    link rel="stylesheet" href="styles.css"

  • p - Paragraph

    Defines blocks of text. Example:

    p>This is a paragraph.

  • pre - Preformatted Text

    Preserves whitespace and formatting. Example:

    pre>Text with spacing

CSS Properties

  • color: Sets the text color.
  • font-size: Defines the font size.
  • font-family: Specifies the font family.
  • background-color: Sets the background color.
  • margin: Defines space around elements.
  • padding: Defines space inside elements.
  • border: Adds a border around elements.
  • text-shadow: Adds shadow effects to text.
  • text-decoration: Modifies text appearance (e.g., underline).
  • alignment: Aligns content horizontally or vertically.
  • line-height: Sets the space between lines of text.
  • justify: Justifies text for alignment.
  • width: Specifies the width of elements.
